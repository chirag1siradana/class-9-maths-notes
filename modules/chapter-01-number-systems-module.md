## 1.1 Natural Numbers, Whole Numbers, and Integers

The journey of numbers starts with **Natural Numbers (ℕ)** = {1, 2, 3, ...}.
Adding zero gives **Whole Numbers (W)** = {0, 1, 2, 3, ...}.
Extending to negatives gives **Integers (ℤ)** = {..., −3, −2, −1, 0, 1, 2, 3, ...}.

**Hierarchy:** ℕ ⊂ W ⊂ ℤ ⊂ ℚ ⊂ ℝ

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Natural → Whole → Integer → Rational → Real│
├─────────────────────────────────────────────────────┤
│  Every set is a subset of the next bigger set.      │
│  📐 Formula : ℕ ⊂ W ⊂ ℤ ⊂ ℚ ⊂ ℝ                    │
│  💡 Example : −5 ∈ ℤ but −5 ∉ ℕ                    │
└─────────────────────────────────────────────────────┘
```


> 🧠 **MEMORY TRICK:** **N-W-I-R-R** → 'Numbers With Increasing Range, Really!' (Natural → Whole → Integer → Rational → Real)



## 1.2 Rational Numbers

A **Rational Number** is any number expressible as **p/q**, where p, q ∈ ℤ and q ≠ 0.

Examples: 3/4, −7/2, 0, 5, 0.333... (= 1/3), 0.142857... (= 1/7)

**Key property:** Rational numbers have either **terminating** or **recurring** decimal expansions.

- Terminating: 3/4 = 0.75 (ends)
- Non-terminating recurring: 1/3 = 0.3̄ (repeats)

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Rational Number                       │
├─────────────────────────────────────────────────────┤
│  A number of the form p/q, p,q ∈ ℤ, q ≠ 0.          │
│  📐 Formula : p/q, q ≠ 0                           │
│  💡 Example : 3/4 = 0.75, 1/3 = 0.333...           │
└─────────────────────────────────────────────────────┘
```


> ⚠️ **BOARD EXAM ALERT:** Terminating decimals → denominator has only 2 and/or 5 as prime factors. This fact is directly asked in board exams!


> ❌ **COMMON STUDENT MISTAKE:** Students write 22/7 = π. This is WRONG! 22/7 is rational (≈ 3.1428...); π is irrational.

## 1.3 Irrational Numbers

An **Irrational Number** CANNOT be written as p/q. Its decimal expansion is **non-terminating and non-recurring**.

Examples: √2 = 1.41421356..., √3 = 1.73205..., π = 3.14159..., e = 2.71828...

**Fact:** √p is irrational if p is a prime number.

**Sum/Product rules:**
- Rational + Irrational = Irrational
- Rational × Non-zero Irrational = Irrational
- Irrational × Irrational = May be Rational! (e.g. √2 × √2 = 2)

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Irrational Number                     │
├─────────────────────────────────────────────────────┤
│  Cannot be expressed as p/q. Non-terminating, non-  │
│  recurring decimal.                                 │
│  📐 Formula : Not of the form p/q                  │
│  💡 Example : √2 = 1.41421356...                   │
└─────────────────────────────────────────────────────┘
```


> 🧠 **MEMORY TRICK:** **INCH** → **I**rrational **N**umbers **C**annot be written as a fraction, they go on and on with **H**idden pattern never repeating!


```
🎓 CLASSROOM MOMENT

  Student : "But sir, the diagonal is √2 and I can't mark it exactly!"
  Teacher : "That's the beauty of irrational numbers — they exist on the number line, but no fraction can pin them down exactly!"
```

## 1.4 Real Numbers and the Number Line

**Real Numbers (ℝ)** = Rational Numbers (ℚ) ∪ Irrational Numbers

Every real number corresponds to a unique point on the number line.

**Locating √n on the number line (Spiral of Pythagoras / Geometrical method):**
1. Draw OA = 1 unit on number line.
2. Draw AB ⊥ OA, AB = 1 unit. Then OB = √2.
3. Draw BC ⊥ OB, BC = 1 unit. Then OC = √3.
4. Continue: OD = √4 = 2, OE = √5, ... etc.

**Successive Magnification:** We can zoom into any part of the number line to locate non-terminating decimals to any desired accuracy.

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Real Numbers                          │
├─────────────────────────────────────────────────────┤
│  ℝ = ℚ ∪ (Irrationals). Every point on number line  │
│  is real.                                           │
│  📐 Formula : ℝ = ℚ ∪ (ℝ \ ℚ)                      │
│  💡 Example : √5 ≈ 2.236 is located geometrically  │
└─────────────────────────────────────────────────────┘
```


> ⚠️ **BOARD EXAM ALERT:** Locating √2, √3, √5 on the number line using right triangles is frequently asked in 2-mark questions!


**Diagram:**

```
  NUMBER LINE — Locating √2 geometrically

  A ──────── B
  |          |
  O ──────── 1

  OA = 1, AB ⊥ OA = 1
  OB = √(1² + 1²) = √2
  Draw arc with centre O, radius OB → hits number line at √2
```

## 1.5 Converting Recurring Decimals to p/q Form

Any recurring decimal can be converted to a rational number p/q.

**Method (from the PDFs):**
Let x = recurring decimal. Multiply by 10ⁿ (n = number of recurring digits) to shift the decimal, then subtract to eliminate the recurring part.

**Example 1:** x = 0.333...
> 10x = 3.333...
> 10x − x = 3 → 9x = 3 → **x = 1/3**

**Example 2:** x = 0.abab... (2-digit repeat)
> 100x = ab.abab...
> 100x − x = ab → 99x = ab → **x = ab/99**

**Example 3:** x = 1.232323...
> 100x = 123.2323...
> 100x − x = 122 → 99x = 122 → **x = 122/99**

**Example 4:** x = 3.1416̄  (only 6̄ repeats)
> 10x = 31.416̄; 100x = 314.16̄
> 100x − 10x = 282.75 → ... *(solve step by step)*

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Recurring Decimal → p/q               │
├─────────────────────────────────────────────────────┤
│  Multiply by 10ⁿ (n = repeating digits) and         │
│  subtract.                                          │
│  📐 Formula : 99x = ab → x = ab/99 for 0.ab̄       │
│  💡 Example : 0.333... = 1/3                       │
└─────────────────────────────────────────────────────┘
```


> 🧠 **MEMORY TRICK:** **10-Subtract-Divide:** Multiply by 10 (or 100 for 2-digit repeat), subtract original, divide → you have p/q!


```
🎓 CLASSROOM MOMENT
  Teacher : "Converting 0.142857142857... to p/q looks scary, doesn't it?"
  Student : "Sir, 6 digits repeat — so we multiply by 10⁶?"
  Teacher : "Exactly! 999999x = 142857, so x = 142857/999999 = 1/7. Magic!"
```


> ⚠️ **BOARD EXAM ALERT:** In board exams, express recurring decimals as fractions in simplest form. Don't forget to reduce p/q by GCD.


> ❌ **COMMON STUDENT MISTAKE:** Students multiply by 10 even when TWO digits repeat — should multiply by 100. Count repeating digits carefully!

## 1.6 Operations on Surds

A **Surd** is an irrational number expressed as ⁿ√a where a is rational and the result is irrational.

**Rules:**
- √(ab) = √a × √b
- √(a/b) = √a / √b
- (√a)² = a
- (√a + √b)(√a − √b) = a − b  *(key identity!)*
- (a + √b)(a − √b) = a² − b

**Addition/Subtraction of Surds:** Only **like surds** can be added/subtracted.
> 3√2 + 5√2 = 8√2 ✓
> 3√2 + 5√3 → cannot be simplified ✗

**Multiplication:**
> √2 × √8 = √16 = 4
> (2 + √3)² = 4 + 4√3 + 3 = 7 + 4√3

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Surds                                 │
├─────────────────────────────────────────────────────┤
│  Irrational roots. Use √(ab)=√a·√b and conjugate    │
│  identities.                                        │
│  📐 Formula : (√a+√b)(√a−√b) = a−b                 │
│  💡 Example : (2+√3)(2−√3) = 4−3 = 1               │
└─────────────────────────────────────────────────────┘
```


> 🧠 **MEMORY TRICK:** **CALM:** **C**onjugate removes surds from denominators, **A**dd only like surds, **L**aws of radicals apply, **M**ultiply surds by combining under one √.


```
🎓 CLASSROOM MOMENT
  Teacher : "Think of surds as siblings — only identical twins (like surds) can be added together."
  Student : "So √2 and √3 are non-identical twins, sir?"
  Teacher : "Exactly! You can't simplify 3√2 + 5√3 just like you can't merge two different people!"
```


> 🏆 **COMPETITIVE EDGE (JEE/NEET Foundation):** Surd operations are the foundation of surds & indices questions in JEE. Master the conjugate identity thoroughly.

## 1.7 Rationalising the Denominator

**Rationalisation** means removing irrational numbers from the denominator by multiplying by an appropriate conjugate.

**Case 1: Monomial surd denominator**
> 1/√2 = 1/√2 × √2/√2 = √2/2

**Case 2: Binomial surd denominator**
> 1/(√a + √b) → multiply by (√a − √b)/(√a − √b)
> = (√a − √b)/(a − b)

**Example (from PDF):**
> 5/(√7 − 2)
> = 5(√7 + 2)/((√7)² − 4)
> = 5(√7 + 2)/(7 − 4)
> = 5(√7 + 2)/3

**Example 2:**
> 1/(√5 + √3)
> = (√5 − √3)/((√5)² − (√3)²)
> = (√5 − √3)/(5 − 3)
> = (√5 − √3)/2

```
┌─────────────────────────────────────────────────────┐
│  📘 CONCEPT:  Rationalisation                       │
├─────────────────────────────────────────────────────┤
│  Eliminate irrational from denominator using        │
│  conjugate multiplication.                          │
│  📐 Formula : 1/(√a+√b) = (√a−√b)/(a−b)            │
│  💡 Example : 1/(√2+1) = √2−1                      │
└─────────────────────────────────────────────────────┘
```


> ⚠️ **BOARD EXAM ALERT:** This is a guaranteed 2–3 mark question in CBSE boards every year. Always simplify fully and verify the denominator becomes rational.

## 1.8 Laws of Exponents for Real Numbers

The laws of exponents apply to all real-number bases (a > 0).

| Law | Formula | Example |
|-----|---------|--------|
| Product | aᵐ × aⁿ = aᵐ⁺ⁿ | 2³ × 2⁴ = 2⁷ |
| Quotient | aᵐ ÷ aⁿ = aᵐ⁻ⁿ | 5⁶ ÷ 5² = 5⁴ |
| Power of Power | (aᵐ)ⁿ = aᵐⁿ | (3²)⁴ = 3⁸ |
| Power of Product | (ab)ᵐ = aᵐbᵐ | (2×3)² = 4×9 |
| Zero Exponent | a⁰ = 1, a ≠ 0 | 100⁰ = 1 |
| Negative Exponent | a⁻ⁿ = 1/aⁿ | 2⁻³ = 1/8 |
| Fractional Exponent | a^(m/n) = (ⁿ√a)ᵐ | 8^(2/3) = 4 |

**Fractional exponent key:**
> a^(1/n) = ⁿ√a
> a^(m/n) = (ⁿ√a)ᵐ = ⁿ√(aᵐ)

**Example (from PDF):**
> (27)^(2/3) = (³√27)² = 3² = **9**
> (64)^(1/6) = ⁶√64 = **2**

```

│  real numbers.                                      │
│  📐 Formula : aᵐ × aⁿ = aᵐ⁺ⁿ; (aᵐ)ⁿ = aᵐⁿ          │
│  💡 Example : 27^(2/3) = (³√27)² = 9               │
└─────────────────────────────────────────────────────┘
```


> 🧠 **MEMORY TRICK:** **PQP-ZNF** → **P**roduct adds, **Q**uotient subtracts, **P**ower multiplies, **Z**ero gives 1, **N**egative flips, **F**raction takes root!

## 📄 Short Answer Questions *(2–3 Marks)*

**Q1.** Rationalise the denominator: 5/(√7 − 2)
**Ans:** Multiply by (√7+2)/(√7+2): 5(√7+2)/(7−4) = **5(√7+2)/3**

**Q2.** Express 0.235̄ as p/q.
**Ans:** Let x = 0.2353535...
1000x = 235.3535..., 10x = 2.3535...
990x = 233 → **x = 233/990**

**Q3.** Simplify: (3 + √2)² − (3 − √2)²
**Ans:** (3+√2)² = 9+6√2+2 = 11+6√2
(3−√2)² = 9−6√2+2 = 11−6√2
Difference = **(11+6√2) − (11−6√2) = 12√2**

**Q4.** Evaluate: (0.125)^(−2/3)
**Ans:** (0.125)^(−2/3) = (1/8)^(−2/3) = 8^(2/3) = (³√8)² = 2² = **4**

**Q5.** Prove that √3 is irrational.
**Ans:** Assume √3 = p/q (lowest terms). Then 3 = p²/q² → p² = 3q² → 3|p → p = 3k.
Then 9k² = 3q² → q² = 3k² → 3|q. Contradiction: p, q have common factor 3. Hence √3 is irrational.

## 📊 Case Study Questions

### Case Study 1: The Tiling Problem

An architect designs a square floor tile. The side of the tile is √2 m. He needs to find the exact perimeter and area, and check if these values are rational or irrational.

**Q1.** Find the perimeter of the tile.
**Ans:** Perimeter = 4 × √2 = **4√2 m** (irrational)

**Q2.** Find the area of the tile.
**Ans:** Area = (√2)² = **2 m²** (rational)

**Q3.** Is the diagonal of the tile rational or irrational?
**Ans:** Diagonal = √((√2)² + (√2)²) = √(2+2) = √4 = **2 m** (rational)

### Case Study 2: Decimal Detective

Teacher gave the class five numbers: 0.25, 0.333..., 1.41421..., 3.14159..., 2.5̄. Students have to classify each and convert recurring ones to p/q.

**Q1.** Classify each number as rational or irrational.
**Ans:** 0.25 = 1/4 (rational, terminating); 0.333... = 1/3 (rational, recurring); 1.41421... = √2 (irrational); 3.14159... = π (irrational); 2.5̄ = 23/9 (rational, recurring)

**Q2.** Convert 2.5̄ to p/q form.
**Ans:** x = 2.555...; 10x = 25.55...; 9x = 23; **x = 23/9**

**Q3.** How many of the five numbers are irrational?
**Ans:** Two: √2 and π

### Case Study 3: Powers in Science

**Q2.** Evaluate: 10¹² ÷ 10⁻⁴
**Ans:** = 10^(12−(−4)) = 10¹⁶

**Q3.** Express 0.000001 as a power of 10.
**Ans:** = 10⁻⁶

## 🎯 Competency-Based Questions

**Q1.** A student claims: 'The sum of two irrational numbers is always irrational.' Is this correct? Give an example to justify your answer.
**Ans:** No, this is incorrect. Counterexample: (2 + √3) + (2 − √3) = 4, which is rational. The sum of two irrationals CAN be rational.

**Q2.** A carpenter measures a wall and says the length is exactly π metres. Can he mark this on a measuring tape? Explain using properties of real numbers.
**Ans:** π is irrational — it has no exact terminating or recurring decimal. The carpenter can only approximate it (e.g. 3.14159 m). However, it CAN be marked geometrically using its definition.

**Q3.** Simplify √(48) − √(27) + √(75) without a calculator. State whether the answer is rational or irrational.
**Ans:** √48 = 4√3, √27 = 3√3, √75 = 5√3. Result = (4−3+5)√3 = **6√3** (irrational)

**Q4.** A square plot has area 2 hectares. Find the exact side length and explain why it cannot be expressed as a terminating decimal.
**Ans:** Side = √2 km (since area = side²). √2 is irrational — its decimal never terminates or repeats.

**Q5.** Evaluate without a calculator: (32)^(3/5) + (81)^(1/4) − (27)^(2/3)
**Ans:** (32)^(3/5) = (⁵√32)³ = 2³ = 8; (81)^(1/4) = ⁴√81 = 3; (27)^(2/3) = (³√27)² = 9. Answer = 8 + 3 − 9 = **2**
