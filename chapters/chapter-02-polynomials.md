# Chapter 2: Polynomials

## Overview

This chapter covers polynomials in one variable, their zeros, and important algebraic identities.

---

## Key Concepts

### 2.1 Polynomials in One Variable
- A polynomial is an expression of the form: **aₙxⁿ + aₙ₋₁xⁿ⁻¹ + … + a₁x + a₀**
- **Degree** of a polynomial: the highest power of the variable.

### 2.2 Types of Polynomials

| Name | Degree | Example |
|------|--------|---------|
| Constant | 0 | 7 |
| Linear | 1 | 2x + 3 |
| Quadratic | 2 | x² − 5x + 6 |
| Cubic | 3 | x³ − 4x |

### 2.3 Zeroes of a Polynomial
- A value **c** is a zero of polynomial p(x) if **p(c) = 0**.
- A polynomial of degree n has at most n zeros.

### 2.4 Remainder Theorem
- When p(x) is divided by (x − a), the remainder is **p(a)**.

### 2.5 Factor Theorem
- (x − a) is a factor of p(x) if and only if **p(a) = 0**.

### 2.6 Factorisation of Polynomials
- Splitting the middle term for quadratics.
- Using algebraic identities.

---

## Important Algebraic Identities

| Identity | Formula |
|----------|---------|
| Square of sum | (a + b)² = a² + 2ab + b² |
| Square of difference | (a − b)² = a² − 2ab + b² |
| Difference of squares | a² − b² = (a + b)(a − b) |
| Cube of sum | (a + b)³ = a³ + 3a²b + 3ab² + b³ |
| Cube of difference | (a − b)³ = a³ − 3a²b + 3ab² − b³ |
| Sum of cubes | a³ + b³ = (a + b)(a² − ab + b²) |
| Difference of cubes | a³ − b³ = (a − b)(a² + ab + b²) |
| Perfect square trinomial | a² + b² + c² + 2ab + 2bc + 2ca = (a + b + c)² |

## Examples

**Example 1:** Find the zero of p(x) = 2x + 4
> 2x + 4 = 0 → x = **−2**

**Example 2:** Check if (x − 2) is a factor of x³ − 8
> p(2) = 8 − 8 = 0 → Yes, it is a factor.

**Example 3:** Factorise x² + 5x + 6
> Find two numbers that multiply to 6 and add to 5: **2 and 3**
> = (x + 2)(x + 3)

---

## Summary

- Degree determines type; zeros satisfy p(c) = 0
- Remainder Theorem: remainder = p(a) when dividing by (x − a)
- Factor Theorem: (x − a) is a factor ⟺ p(a) = 0
- Algebraic identities simplify factorisation and expansion
