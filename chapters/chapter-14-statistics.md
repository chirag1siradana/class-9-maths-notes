# Chapter 14: Statistics

## Overview

This chapter covers the collection, organisation, representation, and analysis of data using statistical measures.

---

## Key Concepts

### 14.1 Data
- **Primary data**: collected directly by the investigator.
- **Secondary data**: collected from existing sources.

### 14.2 Presentation of Data

#### Frequency Distribution Table
- **Ungrouped**: individual values with frequencies.
- **Grouped**: data organised into class intervals.

#### Terms:
- **Class interval**: range of values (e.g., 10–20)
- **Class width/size**: upper limit − lower limit
- **Frequency**: number of times a value/class appears
- **Cumulative frequency**: running total of frequencies

### 14.3 Graphical Representation

| Type | Used For |
|------|---------|
| Bar graph | Discrete data comparison |
| Histogram | Grouped continuous data |
| Frequency polygon | Comparing frequency distributions |

### 14.4 Measures of Central Tendency

#### Mean (Average)
- **Ungrouped data**: Mean (x̄) = (Sum of all observations) / (Number of observations)
- **Grouped data (direct method)**: x̄ = Σfᵢxᵢ / Σfᵢ

#### Median
- Middle value when data is arranged in ascending/descending order.
- For **n** observations:
  - If n is odd: Median = ((n+1)/2)th term
  - If n is even: Median = average of (n/2)th and (n/2 + 1)th terms

#### Mode
- The value that occurs most frequently in the data.

---

## Examples

**Example 1:** Find the mean of: 5, 8, 4, 7, 6
> Mean = (5 + 8 + 4 + 7 + 6) / 5 = 30/5 = **6**

**Example 2:** Find the median of: 3, 7, 1, 9, 4
> Arranged: 1, 3, 4, 7, 9 → n = 5 (odd) → Median = 3rd term = **4**

**Example 3:** Find the mode of: 2, 3, 5, 3, 7, 3, 8, 2
> 3 appears most often (3 times) → Mode = **3**

## Practice Problems

1. Find the mean, median, and mode of: 10, 12, 14, 12, 16, 18, 14, 12.
2. The marks of 10 students are: 45, 50, 38, 62, 70, 55, 45, 38, 62, 45. Find all three averages.
3. Draw a histogram for the following data:

| Class | Frequency |
|-------|-----------|
| 0–10 | 5 |
| 10–20 | 10 |
| 20–30 | 15 |
| 30–40 | 8 |
| 40–50 | 2 |

4. The ages of 6 students are: 12, 13, 14, 13, 15, 14. Find the median.
5. In a data set, mean = 25 and the sum of observations = 500. Find the number of observations.

## Summary

- Data types: primary vs. secondary
- Representations: bar graph, histogram, frequency polygon
- Measures of central tendency: Mean, Median, Mode
- Mean = Σx/n; Median = middle value; Mode = most frequent value
