# Class 9 Maths – Important Formulas

A consolidated reference sheet of all key formulas covered in Class 9 Mathematics.

---

## Chapter 1: Number Systems

| Formula | Expression |
|---------|-----------|
| Product of surds | √a × √b = √(ab) |
| Quotient of surds | √a / √b = √(a/b) |
| Conjugate identity | (√a + √b)(√a − √b) = a − b |
| Rationalisation | 1/(√a + b) × (√a − b)/(√a − b) |
| Power rule | (aᵐ)ⁿ = aᵐⁿ |
| Product rule | aᵐ × aⁿ = aᵐ⁺ⁿ |
| Quotient rule | aᵐ ÷ aⁿ = aᵐ⁻ⁿ |
| Zero exponent | a⁰ = 1 (a ≠ 0) |

## Chapter 2: Polynomials (Algebraic Identities)

| Identity | Formula |
|----------|---------|
| (a + b)² | a² + 2ab + b² |
| (a − b)² | a² − 2ab + b² |
| a² − b² | (a + b)(a − b) |
| (a + b)³ | a³ + 3a²b + 3ab² + b³ |
| (a − b)³ | a³ − 3a²b + 3ab² − b³ |
| a³ + b³ | (a + b)(a² − ab + b²) |
| a³ − b³ | (a − b)(a² + ab + b²) |
| (a + b + c)² | a² + b² + c² + 2ab + 2bc + 2ca |
| a³ + b³ + c³ − 3abc | (a + b + c)(a² + b² + c² − ab − bc − ca) |

## Chapter 12: Heron's Formula

- Semi-perimeter: **s = (a + b + c) / 2**
- Area = **√[s(s−a)(s−b)(s−c)]**
- Equilateral triangle (side a): Area = **(√3/4)a²**

## Chapter 13: Surface Areas and Volumes

### Cuboid (length l, breadth b, height h)
- Lateral SA = 2h(l + b)
- Total SA = 2(lb + bh + hl)
- Volume = lbh
- Diagonal = √(l² + b² + h²)

### Cube (side a)
- Lateral SA = 4a²
- Total SA = 6a²
- Volume = a³
- Diagonal = a√3

### Cylinder (radius r, height h)
- CSA = 2πrh
- TSA = 2πr(r + h)
- Volume = πr²h

### Cone (radius r, height h, slant height l)
- Slant height: l = √(r² + h²)
- CSA = πrl
- TSA = πr(r + l)
- Volume = (1/3)πr²h

### Sphere (radius r)
- SA = 4πr²
- Volume = (4/3)πr³

### Hemisphere (radius r)
- CSA = 2πr²
- TSA = 3πr²
- Volume = (2/3)πr³

## Chapter 14: Statistics

| Measure | Formula |
|---------|---------|
| Mean (ungrouped) | x̄ = Σx / n |
| Mean (grouped, direct) | x̄ = Σfᵢxᵢ / Σfᵢ |
| Median (odd n) | ((n+1)/2)th term |
| Median (even n) | Average of (n/2)th and (n/2+1)th terms |
| Range | Maximum − Minimum |

## Chapter 15: Probability

| Formula | Expression |
|---------|-----------|
| Empirical probability | P(E) = (number of favourable trials) / (total trials) |
| Complement rule | P(not E) = 1 − P(E) |
| Range of probability | 0 ≤ P(E) ≤ 1 |
